# File-Renamer
**A Windows batch script and Command Prompt UI to rename a sequence of files at once**

— Point to a folder that holds your files  
— Set the file format(s) to rename  
— Set exclude options to ignore certain folders  
— Enter a prefix and a starting number  
— View a scan of the files that will be renamed  
— Rename files  

---

### How to launch the FileRenamer UI from any Command Prompt window:

- Create a folder in your Documents called `Scripts`

- Place the `FileRenamer.bat` inside the `Scripts`folder

- Copy the path address of the `Scripts` folder, should be:
  `C:\Users\%USERNAME%\Documents\Scripts\`

- Windows search for Enviorment Variables

- In System Properties click Enviorment Varibles

- In the User Varibles top box select Path > Edit > New

- Paste the `Scripts` folder path you copied

- Select OK > OK > OK to close Enviorment Variables  

Now you can launch the FileRenamer UI just by typing `filerenamer` in any Command Prompt window
